How it works

  1. 1. Each sentence is embedded using all-MiniLM-L6-v2 from Sentence Transformers.
  2. 2. The embeddings are grouped with MeanShift, which estimates cluster groupings directly from the embedding space.
  3. 3. PCA projects the original 384-dimensional embeddings into 2D so you can inspect them visually.
